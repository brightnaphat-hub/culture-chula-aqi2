# Culture Chula — Lobby Display

หน้าจอแดชบอร์ดสำหรับ TV ในล็อบบี้โครงการ CULTURE CHULA

แสดง Real-time AQI สลับกับรูปภาพ/วิดีโอประกาศ-โปรโมชั่นต่างๆ

---

## 📋 วิธีจัดการ Playlist (สำหรับนิติฯ)

### 🟢 เพิ่มรูป/วิดีโอใหม่

**ขั้นตอนที่ 1: อัปโหลดไฟล์**

1. คลิกที่ folder **`media`** (ถ้ายังไม่มี ให้สร้างก่อน — ดูด้านล่าง)
2. คลิกปุ่ม **Add file** → **Upload files**
3. ลากไฟล์รูป (.jpg, .png) หรือวิดีโอ (.mp4) ลงไป
4. เลื่อนลง → กด **Commit changes**

**ขั้นตอนที่ 2: เพิ่มลงใน playlist**

1. คลิกที่ไฟล์ **`playlist.json`** ในหน้าหลัก repo
2. คลิกไอคอน ✏️ (Edit)
3. เพิ่มบรรทัดในรูปแบบ:

```json
[
  { "type": "dashboard", "duration": 30 },
  { "type": "image", "src": "media/promo1.jpg", "duration": 10 },
  { "type": "image", "src": "media/news1.png", "duration": 15 },
  { "type": "video", "src": "media/clip.mp4", "duration": 0 }
]
```

4. กด **Commit changes**
5. รอ 1-2 นาที → จอ TV จะอัปเดตอัตโนมัติ (เว็บเช็คทุก 5 นาที)

---

### 🔴 ลบรูป/วิดีโอ

1. เปิด **`playlist.json`** → ลบบรรทัดที่ไม่ต้องการ → Commit
2. (ถ้าต้องการลบไฟล์ด้วย) เข้า folder `media` → คลิกไฟล์ → ปุ่ม 🗑️

---

### ⏱️ เปลี่ยนเวลาแสดงผล

แก้ตัวเลข `"duration"` ใน playlist.json — หน่วยเป็น**วินาที**

| ค่า | ความหมาย |
|---|---|
| `"duration": 30` | แสดง 30 วินาที |
| `"duration": 10` | แสดง 10 วินาที |
| `"duration": 0` | (เฉพาะวิดีโอ) เล่นจนจบคลิป |

---

## 📐 ขนาดไฟล์ที่แนะนำ

- **รูปภาพ:** อัตราส่วน 16:9 (เช่น 1920×1080) ไม่งั้นจะมีขอบดำสองข้าง
- **วิดีโอ:** .mp4 (H.264), 1080p, ไม่เกิน 30 วินาที, เสียงจะถูกปิดอัตโนมัติ
- ขนาดไฟล์ไม่ควรเกิน **100 MB** ต่อไฟล์ (จำกัดของ GitHub)

---

## 🔄 การอัปเดต

- เว็บจะ**เช็ค playlist ใหม่ทุก 5 นาที**โดยอัตโนมัติ
- AQI จะรีเฟรชทุก **15 นาที**

---

## 📺 วิธีเปิดบน TV

1. เปิด Chrome/Edge
2. ไปที่ URL ของเว็บ
3. กดปุ่ม **⛶ Fullscreen** มุมขวาล่าง (หรือกด F11)

---

## 🆘 ถ้ามีปัญหา

- รูปไม่ขึ้น → เช็คชื่อไฟล์ใน `playlist.json` ว่าตรงกับชื่อไฟล์จริง (case-sensitive)
- วิดีโอไม่เล่น → ต้องเป็น .mp4 (H.264)
- จอค้าง → Refresh หน้า (F5)
